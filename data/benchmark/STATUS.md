extraction artefacts (per-chrom REVEL split, gnomAD tabix extracts,
SpliceAI/PhyloP intermediate VCFs) are regenerated by the build
scripts and were removed in v7 cleanup to free 12+ GB.

Build scripts:

| Script | Purpose |
|--------|---------|
| `build_gnomad_per_chrom.sh` | tabix-extract gnomAD v4 exomes per chrom + sa-build |
| `build_spliceai_phylop.sh` | distill SpliceAI + PhyloP from gnomAD INFO + sa-build |
| `extract_spliceai_phylop_from_gnomad.py` | per-chrom extractor invoked by `build_spliceai_phylop.sh` |
| `clingen_gdv_to_oga.py` | ClinGen GDV CSV → genemap2 TSV |
